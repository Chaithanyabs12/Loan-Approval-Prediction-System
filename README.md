
# Loan Approval Prediction System

A Machine Learning based web application developed using Flask and Scikit-learn to predict loan approval status based on applicant details such as income, loan amount, education, marital status, and credit history.

## Technologies Used
- Python
- Flask
- HTML/CSS
- Scikit-learn
- Pandas
- NumPy
- SQLite

## Features
- User Login & Registration System
- Loan Eligibility Prediction
- Machine Learning Model Integration
- Donut Chart & Result Visualization
- Responsive Modern UI

## Project Modules
- Data Preprocessing
- Model Training
- Loan Prediction System
- User Authentication
- Data Visualization
